### 1. 프로젝트 설치

```bash
$ react-native init --version 0.61.5 testApp
```

방식은 react-native init --version 원하는버전 프로젝트이름

- 버전은 지정해도 되고 안해도 됨



### 2. 화면 시작

```bash
$ npx react-native start
또는
$ npm start
```

그리고 터미널 창을 하나 더 연 뒤

```bash
$ npx react-native run-android //ios경우 run-ios
또는
$ react-native run-andriod
```
